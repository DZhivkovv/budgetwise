## 📊 About

**BudgetEye** is a personal finance application made to help users:

- Track income and expenses
- Organize spending by categories
- Set monthly budgets
- Visualize financial habits through charts
- Gain better control over personal finances

The goal is to make budgeting **not feel like a chore**.

### Backend

- JWT Authentication
